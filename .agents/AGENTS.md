# Identidade Visual CMOC

Este sistema pertence à CMOC, portanto toda a interface deve respeitar a identidade visual da empresa. A paleta de cores deverá ser baseada no logotipo oficial.

## Cores Primárias
- **Azul Escuro (principal)**: `#23005B`
- **Roxo**: `#5C3FA3`
- **Verde**: `#74BE45`
- **Branco**: `#FFFFFF`
- **Cinza Claro**: `#F5F7FA`
- **Cinza Escuro**: `#1F2937`

Estas cores deverão ser predominantes em toda a aplicação.
- Não utilizar vermelho como cor principal.
- Evitar laranja.
- Utilizar o verde apenas para sucesso e indicadores positivos.
- Utilizar o roxo para destaques.
- Utilizar o azul escuro como identidade principal do sistema.

## Aparência
Quero um sistema com aparência de software utilizado por grandes empresas de mineração e indústria. O visual deve transmitir: Tecnologia, Segurança, Robustez, Inteligência, Organização, Precisão, Modernidade.
Não quero um visual genérico. Quero um sistema premium.

## Dashboard
O dashboard deverá lembrar soluções como Microsoft Fabric, Power BI, Azure Portal, Siemens, Schneider Electric, SAP Fiori, IBM Maximo, GE Digital, mas mantendo a identidade visual da CMOC.

## Componentes
Todos os componentes deverão seguir o Design System.
- **Botões**: Azul escuro; Hover roxo; Sucesso verde.
- **Cards**: Fundo branco (tema claro); Fundo cinza escuro (tema escuro); Borda extremamente suave; Shadow elegante; Hover com leve elevação.
- **Gráficos**: Utilizar Azul CMOC, Verde CMOC, Roxo CMOC, Cinzas neutros. Nunca utilizar cores aleatórias.

## Indicadores
KPIs deverão possuir cores inteligentes:
- Produção: 🟢 Verde
- Operação: 🔵 Azul
- Alertas: 🟡 Amarelo
- Crítico: 🔴 Vermelho
- Manutenção: 🟣 Roxo

## Tema Claro
- Visual extremamente limpo, muito branco.
- Cards flutuantes.
- Espaçamento generoso.

## Tema Escuro
- Fundo: `#0B1120`
- Cards: `#111827`
- Menus: `#1E293B`
- Texto: Branco
- Destaques: Azul CMOC, Verde CMOC, Roxo CMOC

## Animações e Ícones
- Animações discretas. Nada exagerado. Tudo deve parecer um software corporativo de alto padrão.
- Utilizar Lucide Icons. Todos os ícones devem possuir o mesmo estilo.

## Logo
O logotipo oficial da CMOC deve aparecer sem distorção e sempre mantendo sua identidade na Tela de Login, Sidebar, Splash Screen, Favicon e Loader.

## Resultado esperado
O sistema deve parecer um software desenvolvido especificamente para a CMOC, com identidade visual consistente, design moderno, experiência intuitiva e acabamento de nível enterprise. Todas as telas, componentes e interações devem seguir a paleta oficial (azul escuro, roxo e verde), transmitindo tecnologia, confiabilidade e sofisticação, sem perder a simplicidade de uso. A interface deve causar impacto visual positivo desde a tela de login até os dashboards operacionais, mantendo um padrão unificado em toda a aplicação.
